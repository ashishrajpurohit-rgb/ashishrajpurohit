UI/UX Testing

UI/UX Testing & Design Validation

Conducted comprehensive UI and UX testing for responsive web applications by validating layouts, typography, spacing, colors, icons, buttons, forms, and user interactions against Figma and Adobe XD designs. Performed responsive testing, cross-browser compatibility testing, accessibility validation, usability testing, and regression testing to deliver a seamless, pixel-perfect user experience.

Project Overview

The Objective

This project showcases my experience in validating responsive web applications from both UI and UX perspectives.

The objective was to ensure every screen matched the approved Figma and Adobe XD designs while delivering a consistent experience across different browsers, operating systems, and devices.

The project included validating typography, spacing, colors, icons, layouts, responsiveness, accessibility, navigation, and usability while collaborating with developers to report defects, verify fixes, and ensure production-ready quality.
